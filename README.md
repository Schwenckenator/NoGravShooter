NoGravShooter
=============

No Gravity Shooter

W = Walk/Fly Forwards<br />
A = Walk/Fly Left<br />
S = Walk/Fly Backwards<br />
D = Walk/Fly Right<br />
Space = Jump/Fly Up<br />
Shift = Crouch/Fly Down<br />
Q = Lean/Rotate Left<br />
E = Lean/Rotate Right

Mousebutton1 = Shoot<br />
Mousebutton2 = Scope (when applicable)<br />
Mousewheelup = Previous Weapon<br />
Mousewheeldown = Next Weapon

R = Reload<br />
G = Grenade<br />
F = Interact<br />
K = Suicide

1 = Laser Rifle<br />
2 = Assault Rifle

3 = High Intensity Beam<br />
4 = Sniper Rifle

5 = Shock Cannon<br />
6 = Shotgun

7 = Plasma Blaster<br />
8 = Rocket Launcher

9 = Gravity Mines<br />
0 = Frag Grenades
